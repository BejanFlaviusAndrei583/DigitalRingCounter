# Digital Ring Counter PCB Design

## About

This project was completed as part of a university electronics course.

The objective was to design a digital circuit and create a printed circuit board (PCB) using electronic design software.

## What I Worked On

- Circuit schematic
- PCB layout
- Component placement
- PCB routing
- Power supply section
- Digital logic circuit implementation

## Components Used

- 74HCT164
- 74HCT132
- 74HCT32
- LM7805 Voltage Regulator
- LEDs
- Push Buttons
- Resistors
- Capacitors

## Tools

- Proteus / KiCad / Eagle *(alege programul pe care l-ați folosit)*
- PDF Export

## What I Learned

- Reading electronic schematics
- PCB design basics
- Component placement
- PCB routing
- Digital logic circuits
- Basic power supply design
